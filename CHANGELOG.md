# Changelog

All notable changes to Bridge Clearance Calculator will be documented in this file.

## [0.1.0.0] - 2026-03-18

### Added
- Global bridge clearance calculator with support for worldwide bridges
- Interactive Leaflet map with bridge markers, search, and geolocation
- NOAA CO-OPS tide provider for US coastal bridges
- LINZ tide provider for New Zealand bridges
- Rule of Twelfths tidal interpolation engine for accurate clearance estimates
- Custom bridge support with automatic nearest tide station detection
- Safe passage window finder showing all viable crossing times per day
- Single-time clearance check with safe/caution/danger status
- Bridge span selector for multi-span bridges
- Add/remove custom bridges with localStorage persistence
- Design system (DESIGN.md) — Satoshi + DM Sans typography, maritime color palette, component patterns
- Project planning documents (TODOS.md) — prioritized backlog from CEO and design reviews
